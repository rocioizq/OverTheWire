# OverTheWire Writeups

Este repositorio contiene mis apuntes, pasos y soluciones personales de los wargames de **[OverTheWire](https://overthewire.org/)**.  
El objetivo es documentar mi progreso en los diferentes niveles, explicar los comandos aprendidos y crear un portfolio de práctica en ciberseguridad.

Cada carpeta contiene un `README.md` con:
- 🎯 **Objetivo del nivel**  
- 📝 **Pasos seguidos para resolverlo**  
- 💻 **Comandos utilizados**  
- 📚 **Conceptos o herramientas aprendidas**

## Este repositorio funciona como:
- Un **diario de aprendizaje** en ciberseguridad  
- Un **portfolio** para demostrar práctica real en Linux y seguridad  
- Una forma de consolidar lo aprendido mientras avanzo
